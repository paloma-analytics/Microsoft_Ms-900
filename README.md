# 🎯 O que é o Microsoft MS-900?

O **MS-900** é a certificação fundamental da Microsoft que valida o conhecimento sobre os serviços e soluções do **Microsoft 365**.  

Mais do que uma prova, ela representa um mapa completo de como as organizações modernas trabalham: como os usuários são criados e gerenciados, como os arquivos são armazenados e compartilhados com segurança, e como as equipes colaboram no dia a dia.

---

# 🤔 Por que essa apresentação existe?

Mesmo que você já use o **Teams**, o **OneDrive** ou o **SharePoint** no seu trabalho, provavelmente nunca parou para pensar em como tudo isso funciona nos bastidores.

- Quem cria os usuários?  
- Como o acesso é controlado?  
- Como um arquivo chega com segurança até um parceiro externo?  

Esta apresentação responde essas perguntas de forma prática e objetiva.

---

# 📌 O que você vai encontrar aqui

## 👤 Gestão de Identidade
Criação e gerenciamento de usuários no **Azure Active Directory (Azure AD)**.

## ☁️ Armazenamento em Nuvem
Como funciona o armazenamento e sincronismo com o **OneDrive**.

## 💬 Colaboração Corporativa
Criação de equipes, canais e comunicação no **Microsoft Teams**.

## 🗂️ Gestão de Conteúdo
Estrutura de sites, permissões e bibliotecas no **SharePoint Online**.

---

# 💡 Conceito Principal

Tudo começa pelo mesmo ponto de qualquer ambiente Microsoft:

> Saber **quem são os usuários** e **o que cada um pode acessar**.

Essa é a base para segurança, produtividade e colaboração dentro do Microsoft 365.
